## Билет 29. Sort
Напишите аналог утилиты `sort`.
### Функциональность
- Поддерживаемые опции:
    * `--ignore-leading-blanks`
    * `--dictionary-order     `
    * `--ignore-case          `
    * `--general-numeric-sort `
    * `--ignore-nonprinting   `
    * `--numeric-sort         `
    * `--reverse              `
- Ввод данных с консоли или входного файла.
- Исходные данные могут __не поместиться__ в памяти.
### Технологии
- I/O
- Collections Framework
