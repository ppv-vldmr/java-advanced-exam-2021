## Билет 25. TODO List
Сделать приложение, реализующее список задач.
### Функциональность
- Загрузка списка задач из файла.
- Запись списка задач в файл.
- Операции с задачами
    * добавление задачи;
    * удаление задачи;
    * пометка задачи как выполненной.
- Просмотр задач:
    * всех;
    * выполненных;
    * невыполненных.
- Интерактивный ввод-ввод осуществляется с консоли.
### Технологии
- I/O
- Collections Framework
